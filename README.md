# Gatsby Playground

Gatsby + Netlify + Gatsby-Netlify-CMS 공부용 Repo.

## Install

### Environment

> OS : Windows10 x64, macOS 12.3.1
> Node : v16.14.2(LTS)
> Yarn : v1.22.18

### Dependencies

```bash
yarn install # 또는 그냥 yarn
```

## Dotenv

```bash
GATSBY_CONCURRENT_DOWNLOAD # 15로 설정. 기본값은 200 (줄이지 않으면 RAM 용량 초과로 실행 안됨)
```

## Run

```bash
yarn develop # 또는 yarn start
```
